Spite Level Editor Wiki

The official wiki for the Spite Level Editor.

The Spite Level Editor is a public level editor for ULTRAKILL. Where you can upload your level into Envy Level Loader at any time, no matter the quality. If its first time using Spite then click the 'Start Tutorial' button to start the tutorial, where you will learn how to create levels, from start to finish. Or if you want to see documentation on a certain topic or on something specific, use the panel above.

It is important to note that even if this wiki is for Spite LE, information given here can also be applied to the Rude Level Editor, however, Rude LE has some differences.

The wiki website has been created by Alma and the entries of the wiki have been created by Alma, Doomah and by our contributors. If you wish to contribute into the wiki, you can go into this page for more information.
